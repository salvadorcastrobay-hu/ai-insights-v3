```
--brand/400                  → #6f93eb  (Humand 400)  — acento de marca
--blueprimary/100            → #eff2ff                — fondo de tabla header
--blueprimary/800            → #213478                — texto sobre fondo blueprimary/100
--background/layout/default  → #f5f6f8  (Neutral 50)  — fondo de página
```
