Ask This Chart Without Using a Dropdown

Two low-friction interaction patterns for the dashboard. Both keep the question grounded in the chart context, like Top 10 Pains → Manual Processes, without forcing the user into a selectbox flow.

anchored to chart context minimal visual noise
Option A • Button on the chart card

Open a focused pop-up from the chart itself

The action lives exactly where the user is looking. Clicking it opens a light modal with the chart already selected, quick prompts, and a free-text input.
